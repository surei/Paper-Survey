# 論文調べ

[論文の読み方などのまとめ](how_to_read_study.md)
取り敢えず、AbstractとIntroduceら辺だけ読む。

## ルール
1. Issues読みたい論文をIssuesに投稿
2. Abstractだけ読む。(英語は和訳する(英語が苦手なので))
3. ラベル付け(精読するかなど)

## ラベル
- 投稿日
- ML(Machine Learning)
- DL(Deep Learning)(フォーカスとして)
- MP(Menu Planning)
- Intensive reading(精読)
- Extensive Reading(多読)
